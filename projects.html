Online Restaurant Management System

A comprehensive system built with MySQL for database management, a responsive frontend using HTML, CSS, and JavaScript, and a PHP-powered backend. Features include order processing, delivery route optimization, and secure user authentication.

Comparative Analysis of Cryptographic Algorithms

A project evaluating the encryption and decryption times of AES-128, 3DES, Blowfish, and RSA across various data types and sizes in a virtual machine environment. The study highlights AES-128's consistent balance between security and performance, with ECB and CTR modes being the fastest.
